# 🎨 HGR Studio PRO - Editor de Adesivos

O **HGR Studio PRO** é uma ferramenta desenvolvida em Python voltada para a criação e edição personalizada de stickers (adesivos). O foco do projeto é oferecer uma interface funcional para manipulação de imagens, permitindo desde a geração até o tratamento fino de elementos visuais.

## 🚀 Funcionalidades Principais

*   **Manipulação de Imagens:** Processamento eficiente de arquivos de imagem para criação de stickers.
*   **Apagador Manual:** Ferramenta de edição estilo "canva", permitindo a remoção precisa de partes da imagem para refinamento do adesivo.
*   **Sistema de Duplicação:** Facilidade para replicar elementos e otimizar o fluxo de trabalho.
*   **Gerenciamento de Arquivos:** Sistema integrado para salvar e organizar as criações diretamente pelo software.

## 🛠️ Tecnologias Utilizadas

*   **Linguagem:** [Python](https://www.python.org/)
*   **Interface e Gráficos:** Bibliotecas de processamento de imagem (Pillow) e interface gráfica.
*   **Lógica de Programação:** Implementação de algoritmos de edição e manipulação de matrizes de imagem.
